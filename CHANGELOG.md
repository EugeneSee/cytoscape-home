## [1.0.0] - 2025-12-04

### Added

- Initial stable release of Cytoscape Home
- Modern React 18 application with Vite build system
- Integration with Cytoscape.js for graph visualization
- TanStack React Query for efficient data fetching
- Tailwind CSS for responsive styling
- Framer Motion for smooth animations
- NDEx client integration for network data
- Search functionality with MiniSearch
- Multiple application wizards (Gene, Network, Enrichment, Tutorial)
- Comprehensive UI components library
- Authentication layout and pages
- Main application layout with navigation
- Responsive design optimized for all devices

### Infrastructure

- `.gitignore` configured for Node.js, build artifacts, and editor files
- `.gitattributes` for proper text/binary file handling and line endings
- `.editorconfig` for consistent code formatting across editors
- ESLint configuration for code quality
- Tailwind CSS and PostCSS configuration
- Vite configuration with React plugin
