# Cytoscape Home

## Project Overview
Cytoscape Home is the official web platform for the Cytoscape ecosystem.
It serves as a central gateway for researchers and scientists to explore
Cytoscape tools for biological network visualization and analysis.

The application is built using React and Vite to ensure high performance,
maintainability, and a modern user experience. It integrates with several
Cytoscape ecosystem tools such as NDEx IQuery, GeneMANIA, EnrichmentMap,
and WikiPathways.
